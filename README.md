# CV Dynamic Builder — Editor Modular de Currículums

**CV Dynamic Builder** es una aplicación web interactiva de alto rendimiento diseñada para la creación, personalización y previsualización de currículums en tiempo real. Este proyecto destaca por su enfoque en la **modularidad extrema** y la **escalabilidad**, permitiendo incorporar nuevas plantillas y características de manera declarativa sin modificar la lógica del motor principal.
